Personal repository. Working on my scientific iniciation on eccentricity of heavy ions collisions.

If you want to run the code:

1. Be sure to install ROOT FRAMEWORK from ROOT official website (https://root.cern/install/);
2. The files need to be in the same folder;
3. Run collisionsDraw.cpp first, then run eccen.C. Because of a ROOT bug, it may be necessary to exit ROOT and do this step again.
